Switches Dials Knobs

Switches Dials Knobs is a modular, interactive application built with the Observable Framework. It provides a unique interface for crafting dynamic controls like switches, dials, and knobs, complete with customizable SVG-based visuals.

This project is designed to showcase how visual interactivity can be combined with state-driven behavior to create compelling user interfaces. Whether you're building dashboards, control panels, or just experimenting with creative designs, Switches Dials Knobs offers flexible components to bring your ideas to life.

  • SVG Customization: Every control is rendered using scalable vector graphics for crisp visuals.

SVG layers

Shadows (multiply) create depth and grounding Highlights (screen) add dimension and surface detail Glows (plus-lighter) suggest illumination or importance Texture (color-burn) adds material qualities

[id^="shadow-"] { mix-blend-mode: multiply; opacity: 0.5; }

[id^="highlight-"] { mix-blend-mode: screen; opacity: 0.7; }

[id^="glow-"] { mix-blend-mode: plus-lighter; filter: blur(5px); }

[id^="texture-"] { mix-blend-mode: color-burn; opacity: 0.3; }

SVG Layer & Animation Conventions

Layer Structure

  • Groups (<g>) define visual effects via blend modes
  • Child elements define animation targets via classes

Naming Schema

<!-- Effect Groups -->
<g id="retro-shadow">
<g id="retro-glow">
<g id="retro-light">

<!-- Animation Classes -->
